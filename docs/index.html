Trim multi-line commands automatically.

Trimmy watches your clipboard and turns wrapped shell snippets into a single clean line, so they paste and run exactly once.

Download latest Read the docs
  • Auto-trim when it looks like a command (respects \ continuations).
  • One-click “Paste Trimmed” + “Paste Original” with global hotkeys.
  • Preview shows what will paste, and what gets removed.

Smart cleanup

Strips prompt gutters, optional box-drawing chars, and keeps blank lines when you want them.

On-device

No network calls. Everything happens locally on your Mac.

CLI

Includes a headless trimmer for scripts: swift run TrimmyCLI.
